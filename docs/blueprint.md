# **App Name**: ClaimFlow AI

## Core Features:

- User Authentication: Secure registration and login system with user profile management.
- AI Chatbot Guidance (ClaimCoach): An AI chatbot that guides users through the claim submission process by asking follow-up questions and suggesting required evidence. LLM acts as a tool.
- Evidence Upload: Secure system for uploading damage photos and supporting documents with progress tracking.
- AI Damage Detection (SnapClaim): AI analyzes uploaded images to detect and summarize visible vehicle damage.
- OCR Auto-Fill: Extracts text from uploaded documents to auto-populate claim form fields.
- Fraud Risk Scoring (ClaimGuard): Rule-based fraud detection system that analyzes image metadata, documents, and inconsistencies to output a risk score.
- Claim Decision Dashboard: Displays claim status, damage analysis results, and fraud risk score with AI explanations.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke trust and reliability.
- Background color: Very light blue (#F0F8FF) for a clean and professional look.
- Accent color: Pale Green (#90EE90) for positive feedback and approval indicators.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text.
- Use clean, modern icons to represent different claim statuses and actions.
- Maintain a clean, step-by-step layout with progress indicators for claim submissions.
- Subtle animations for loading states and successful actions.
